El autor del trabajo es **Lucas Osejo Martínez** (lucasosejo2@gmail.com).

El título del trabajo es **"Inflación en Colombia comparado con la Región: Un Manual para Inversores"**. 

En este estudio se analiza la estabilidad monetaria de las principales economías de Sudamérica (Argentina, Brasil, Chile, Colombia y Perú). A través de la visualización de datos con mapas y gráficos interactivos, se explora la evolución de los precios, la correlación entre inflación y crecimiento del PIB, la resiliencia ante el shock del COVID-19 y el compromiso de cada país con la innovación (I+D), con el fin de determinar cuál es el mercado más atractivo para la inversión extranjera.
